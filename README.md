# easyadmin

#### 介绍
为了快速搭建项目框架进入开发,所以打算做一个EasyAdmin作为后台的脚手架,节省时间。



#### 功能规划清单

1.  后台用户管理
2.  部门管理
3.  菜单管理
4.  角色管理
5.  字典管理
6.  权限分配
7.  配置管理
8.  登录日志
9.  操作日志
10. 登录日志
11. 定时任务
